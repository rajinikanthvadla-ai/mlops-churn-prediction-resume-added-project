# End-to-End MLOps Pipeline — Customer Churn Prediction

## Overview

Production-grade MLOps pipeline using:

- AWS SageMaker Pipelines
- GitHub Actions CI/CD
- EKS deployment
- MLflow tracking
- Drift detection with Evidently
- FastAPI backend
- Streamlit frontend

## Architecture

GitHub → SageMaker Pipeline → Model Registry → Endpoint → EKS → FastAPI → Streamlit → MLflow → Drift Detection
